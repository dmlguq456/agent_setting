# 협업·소통 스타일 (collaboration style)

사용자와의 작업 흐름·소통·피드백 패턴. cross-project 적용.

## 사용자 수동 메모

- **응답 가독성 (강한 선호, 2026-06-15)**: 답변을 짧고 평범한 문장으로 쓴다.
  - 한 줄에 가운뎃점(·), 화살표(→), 세로바(|), ⓐⓑ, 괄호 약자를 빽빽이 욱여넣는 밀도 높은 표기 금지.
  - 모바일(remote-control)에서도 읽히게: 짧은 문장 + 여백.
  - 표는 정말 비교가 필요할 때만. 비표준 약자는 풀어서.
  - 핵심 먼저, 부연 최소. 한 문단에 한 생각.
