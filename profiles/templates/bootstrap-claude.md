## Claude masked-profile attach layer

This config home exposes only the selected Skills, agents, guard settings, and
specialization fragment. The dispatch prompt supplies the canonical portable
worker kernel and exactly one declared worker-type fragment; do not reread the
four main core documents or a full adapter bootstrap unless the assignment
names a specific section as required input.

`CLAUDE_CONFIG_DIR` attaches this partial projection. Runtime-owned project or
user `CLAUDE.md` inheritance may still occur and is not represented as physical
masking by this profile.
