## Codex masked-profile attach layer

This config home exposes only the selected Skills, agents, guard settings, and
specialization fragment. The dispatch prompt supplies the canonical portable
worker kernel and exactly one declared worker-type fragment; do not explicitly
read the main Codex adapter bootstrap unless the assignment names a specific
section as required input.

`CODEX_HOME` attaches this partial projection. Runtime-owned project
`AGENTS.md` discovery may still occur and is not represented as physical
masking by this profile.
