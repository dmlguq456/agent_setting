# Paper Architecture Figures: Layout Guidance Only

Policy established 2026-05-28.

Paper architecture diagrams in the user's established deck style exceed the reliable craft boundary of LLM-generated visuals. The design-team does not produce the final diagram. It provides only:

- **Composition and layout guidance:** block list with labels, role colors, positions, flow, hierarchy, and emphasis; output as a Markdown sketch or wireframe-grade SVG.
- **Reference pointers:** `<agent-home>/user_profile/assets/figure/svg/` for objects extracted from PPTX, `figure_ppt/*.pptx` for editable originals, and Part B of `mem profile 01_paper_figure_style` for macro style.
- **A user-completion path:** duplicate slide shapes in PPTX, then replace labels and colors.

This applies only to paper architecture figures. UI, web apps, slide HTML, icons, and Mermaid or Excalidraw diagrams may be completed by the LLM through the normal visual-verification loop.
